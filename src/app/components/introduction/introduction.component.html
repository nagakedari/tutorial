Introduction

This tutorial is to provide detail knowledge about Ethereum blockchain and Solidity development. The entire tutorial is structured in to three main areas which are Ethereum and blockchain fundamentals, Building smart contracts using solidity and web applications interaction with blockchain and smart contracts using tools like web3.js.


Why should you take this tutorial?

  1. The blockchain technology is in very early adopt stage and it has a huge demand for blockchain developers already.
  2. If you want to build smart contracts or build a web appliation which require blockchain interaction, this tutorial provides all the fundamentals you need to know.

Pre-Requisite knowledge

You need to have basic knowledge in any one of these: Web Development, JavaScript, Ajax-Requests. You DO NOT need to know cryptography.
