Working with GETH


Download and Install Geth: Download the geth from Download Geth and install on to your machine.


Running Geth: To run geth in windows, open powershell and type geth command. In linux or mac, run the geth command by opening a terminal. In this tutorial we will use windows OS.



When you run geth command for the first time, geth runs in fast sync mode (to understand sync modes, refer How it Works section) and downloads the blocks on to your local machine. In subsequent runs, geth will run in full sync mode. To stop a running geth, use Ctrl+C command. The path where the blocks gets stored in our computer varies based on OS.

  • Windows - ~home_directory\AppData\Roaming\Ethereum
  • Linux - ~home_directory/.ethereum
  • Mac Os - ~home_directory/Library/Ethereum

Ethereum folder has geth and key sub folders. geth folder contains all the block data where as key folder contains the keys. If you want to run the geth in Fast scan mode, you need to delete chaindata folder which is under geth folder.
